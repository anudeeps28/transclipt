# Frontend Design Skills (Optional)

This folder contains design-focused skills for frontend work: brand guidelines, UI styling, slide decks, design systems, and canvas-based design.

**This is optional.** If your project doesn't involve frontend design or visual work, you can safely delete this entire folder to reduce the install size (~200 files, mostly fonts and CSV data).

## To remove during install

Delete before or after running the installer:

```bash
rm -rf skills/frontend-design/
```

Or after install:

```bash
rm -rf ~/.claude/skills/frontend-design/    # global install
rm -rf .claude/skills/frontend-design/       # project install
```

## What's included

| Subfolder | Purpose |
|---|---|
| `banner-design/` | Banner and header image generation |
| `brand/` | Brand guideline enforcement and color extraction |
| `design/` | Logo, icon, and visual design patterns |
| `design-system/` | Design token generation and validation |
| `slides/` | Slide deck creation and layout |
| `ui-styling/` | Canvas fonts and UI styling rules |
| `ui-ux-pro-max/` | Advanced UI/UX patterns |
