Why transclipt?

⚡

4x faster than Whisper

Built on faster-whisper (CTranslate2), not the original OpenAI Whisper. Same accuracy, fraction of the time and memory.

🔒

100% local and private

Everything runs on your machine. No data leaves your computer, no API keys needed, no usage limits. Free forever.

🌐

1000+ sites supported

YouTube, Instagram, TikTok, Twitter, Spotify, Reddit, Vimeo, Facebook and everything else yt-dlp supports.

📄

Multiple output formats

Plain text, Markdown, SRT subtitles, or structured JSON. Pick the format that fits your workflow.

🎧

Auto language detection

Whisper automatically detects the spoken language. Force a specific language with --language for better accuracy.

🛠

Batch processing

Pass multiple URLs in one command. Each gets its own output file, auto-named from the video title.

Works with everything

Any platform yt-dlp supports — and that's a lot.

YouTube Instagram Reels TikTok Twitter / X Spotify Reddit Vimeo Facebook SoundCloud + 1000 more

Output formats

.txt

Plain text — clean transcript, nothing else

.md

Markdown — with metadata header and formatting

.srt

Subtitles — timestamped, ready for video editors

.json

Structured — segments with timestamps for processing
